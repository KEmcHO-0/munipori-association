Monipuri Student Welfare Association — Website

Simple, responsive website scaffold for the Monipuri Student Welfare Association (Alumni).

Features
- Multi-page responsive frontend (Home, About, Events, Members, Gallery, Contact)
- Contact, registration and newsletter endpoints (Node.js + Express)
- Data persisted to JSON files in `/data/` (easy to replace with MongoDB/MySQL)

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run server:

```bash
npm start
# or for development with auto-reload (if you have nodemon): npm run dev
```

3. Open http://localhost:3000 in your browser.

Notes
- This is a beginner-friendly scaffold. Replace the JSON storage with MongoDB or MySQL as needed.
- Update site copy, images, and the logo in `public/assets/` as desired.
